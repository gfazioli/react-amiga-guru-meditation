# Get Started

## Requirements

* [node](https://nodejs.org)
* [yarn](https://yarnpkg.com/en/docs/install#mac-stable)

## Installation

You can use either **npm**

```bash
npm i react-amiga-guru-meditation
```

or **yarn**

```bash
yarn add react-amiga-guru-meditation
```
